# url2mdlink

## Overview

**url2mdlink** convert a URL to its markdown link.


## Installation

``` bash
$ go get github.com/takatori/url2mdlink
```

## Usage

``` bash
$ url2mdlink https://github.com/takatori/url2mdlink
[takatori/url2mdlink](https://github.com/takatori/url2mdlink)
```
